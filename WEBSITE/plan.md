

# Little Wonders Gifts — Презентационен сайт (БГ)

## Технологии
React + Vite + TypeScript + Tailwind CSS + React Router (вече налични). Добавяме `react-helmet-async` за SEO мета тагове.

## Цветова гама (Tailwind токени)
- **baby-blue**: `#A8D8EA` (основа/фон)
- **baby-blue-light**: `#D6EFF8` (секции)
- **rose**: `#F472B6` (CTA/акценти)
- **rose-dark**: `#EC4899` (hover)
- **white**: `#FFFFFF` (основни площи)
- **pastel-yellow**: `#FEF3C7` (допълнение)
- **pastel-mint**: `#D1FAE5` (допълнение)
- **pastel-lilac**: `#EDE9FE` (допълнение)
- **text-dark**: `#1E293B`
- **text-muted**: `#64748B`

## Типография
- Заглавия: **Nunito** (Google Fonts, заоблен и приятелски)
- Текст: **Inter** (четим)
- Увеличен line-height и spacing

## Дизайн стил
- Пастелни фонове с blob/wave SVG форми
- Rounded компоненти (rounded-2xl/3xl), меки сенки
- Декоративни SVG иконки: звездички, сърчица, облачета (inline, леки)
- Hover micro-interactions: bounce, scale на карти
- Лек background pattern (точки/звездички) чрез CSS
- Sticker-style карти с леки сенки и border

## Страници

### 1. Начало (`/`)
- **Hero**: H1 "Сладки подаръци, създадени с любов", подтекст, 2 CTA бутона ("Разгледай галерията", "Пиши ни"), декоративни облачета/звездички
- **Badge pills**: "Ръчна изработка", "Персонализация", "Подходящо за бебета"
- **Какво правим**: 4 карти с иконки (подаръци, керамични фигури, клипсове за биберон, подаръчни комплекти)
- **Популярни продукти**: Grid 6-9 placeholder снимки с hover zoom
- **Защо да избереш нас**: 4 sticker-style карти
- **Отзиви**: 3 карти с имена, звезди, текст
- **Финален CTA банер**: розов фон, текст + бутон "Свържи се с нас"

### 2. Галерия (`/galeriya`)
- Филтри (chips): Всички, Керамични фигури, Клипсове за биберон, Подаръчни комплекти
- Grid с placeholder изображения + hover ефект
- Lightbox за преглед на снимки
- SEO текст (2-3 параграфа) на български

### 3. За нас (`/za-nas`)
- История на бранда (топъл, семеен тон)
- "Нашите ценности" — 3 карти с иконки
- "Как създаваме подаръците" — 4 стъпки с илюстрации
- "Материали и безопасност" — секция за бебешка безопасност

### 4. Услуги (`/uslugi`)
- 4 карти с описание и CTA: Персонализирани подаръци, Керамични фигури по поръчка, Клипсове за биберон с име, Подаръчни комплекти
- FAQ акордеон (5-7 въпроса на български)

### 5. Контакти (`/kontakti`)
- Форма: Име, Имейл, Съобщение (на български)
- Социални икони
- Приятелско съобщение: "Ще ти отговорим възможно най-скоро 💗"
- Placeholder за Google Maps (lazy load)

## Общи компоненти
- **Header/Navbar**: Лого + навигация на български, мобилно меню (hamburger)
- **Footer**: Лого, навигация, социални икони, копирайт — всичко на български
- **SEO**: `react-helmet-async` за мета тагове, OG/Twitter cards, JSON-LD schema за всяка страница
- **ScrollToTop**: при навигация между страници

## Анимации (леки)
- CSS transitions за hover (scale, bounce)
- Float анимация за декоративни иконки
- Fade-in при scroll (CSS-only или Intersection Observer)
